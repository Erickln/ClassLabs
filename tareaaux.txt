A. Programación en Java

1. Terminar la función de reloj

Completa una función en Java que simule el comportamiento de un reloj en formato de 24 horas.

El reloj debe:

Aceptar horas, minutos.

Validar que los valores estén dentro del rango (00–23 para horas, 00–59 para minutos y segundos).

Poder avanzar el tiempo (sumar horas y minutos).


Restricción: No uses funciones de fecha/hora ya hechas de Java (LocalTime, Calendar, etc.).

B. Uso de GitHub

1. Documentación de cambios

Cada archivo que crees o edites debe tener comentarios explicativos en el código.

Cada vez que hagas commit, escribe un mensaje claro que explique el cambio realizado.

2. Subida a GitHub

Asegúrate de hacer add, commit y push de todos los archivos creados/modificados al repositorio de GitHub antes de la clase.
